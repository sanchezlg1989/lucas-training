# 🏋️ Plan de Entrenamiento — Lucas Sanchez

**Mesociclo 1 · Fuerza de Base · 4 semanas**

Web app para seguimiento del plan de entrenamiento personalizado.

## Estructura

```
├── index.html   # App principal
├── style.css    # Estilos
├── app.js       # Lógica y datos
└── vercel.json  # Config deploy
```

## Funcionalidades

- 4 semanas (Acumulación, Intensificación, Choque, Descarga)
- 3 días por semana
- Registro de pesos por ejercicio (guardado en localStorage)
- Notas por día
- Diseño responsive para móvil y desktop

## Deploy en Vercel

1. Subir a GitHub
2. Importar el repo en [vercel.com](https://vercel.com)
3. Deploy automático ✓
